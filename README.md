# Traffic Accidents Analysis (EDA & Visualization)

## Overview

This project performs exploratory data analysis (EDA) on a traffic accidents dataset to identify patterns, trends, and key factors contributing to accidents.

## Objectives

* Analyze accident distribution over time
* Identify key factors influencing accidents
* Visualize trends using graphs and charts

## Tools Used

* Python
* Pandas
* numpy
* Matplotlib / Seaborn
* Google Colab

## Project Structure

* `EDA_traffic_accidents_analysis.ipynb` → Main notebook containing analysis and visualizations
